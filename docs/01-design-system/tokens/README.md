# Tokens

Design tokens for the Dutchy Design System: consistent foundations for color, type, spacing, and effects.

## Sections
- [Colors](./colors.md) — Palette, semantic colors, themes, contrast guidance
- [Typography](./typography.md) — Font stacks, type scale, headings, labels
- [Spacing](./spacing.md) — 4px-based scale, layout spacing, grid gaps
- [Effects](./effects.md) — Borders, shadows, transitions

## Quick Notes
- Zero-radius philosophy; emphasis on borders over shadows
- High-contrast defaults with alternate themes and dark mode
- Type uses Space Grotesk (display), Inter (body), JetBrains Mono (code)
